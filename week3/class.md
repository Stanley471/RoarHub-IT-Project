## CSS elements
- "display: flex": This turns an element into a flex contaier makes every child item arrange themselves in a row(left to right by default)
### Key Properties of "display: flex"
- flex-direction: this is the direction of how child items arrange themselves. the values are either row(default) or column.
- align-items: this defines the position of the items along the vertical axis
- justify-content: this defines the psoition of the items along the horizontal axis
- gap: this gives space between items 

## Universal Selector
"*" is used as a universal selector to style every element on the page.
The most common styles in universal selectors include.
- margin: 0
- padding: 0
- box-sizing: border-box
